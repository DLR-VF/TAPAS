String de.dlr.ivf.tapas.tools.fileModifier.TPS_ParameterFileConverterGUI.showModal ( final JFrame  parent,
String  simKey 
)

Zeigt ein Fenster zum Anzeigen der Config für den übergebenen Simulationskey. Zurückgegeben wird der Simulationskey, der zuletzt bearbeitet wurde, so dass diese Config in dem Parent-Frame angezeigt werden kann. Die Änderungen die an der Config vorgenommen wurden sind in den Enums ParamFlag, ParamValue und ParamString gespeichert. Sollte ein Simulationskey zurückgegeben werden, der vorher noch nicht vorhanden war, sollte daraus eine neue Simulation gestartet werden.
