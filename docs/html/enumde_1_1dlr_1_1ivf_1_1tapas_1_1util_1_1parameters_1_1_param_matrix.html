Detailed Description

This class provides all matrix enums which determine the name of parameters available in the application

This enum provides constants in matrix form for the application. It is possible that on constant has two matrices depending on the type of the constant. If it is simulation type dependent there exist one matrix for the base an one for the scenario case.
