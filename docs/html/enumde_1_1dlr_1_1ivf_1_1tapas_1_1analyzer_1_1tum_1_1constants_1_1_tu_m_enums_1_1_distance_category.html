static double [] de.dlr.ivf.tapas.analyzer.tum.constants.TuMEnums.DistanceCategory.createFilteredDistances ( boolean[]  filter)
static

Creates new distances from a filter according to the default classes in DistanceCategory.
Example for default classes [<1, 1-3,3-5, 5-7,7-10,10-25,25-100,>= 100] and filter [F,F,T,F,F,T,F,F] becomes [<5, 5-25, >= 25,-,-,-,-,-]

Parameters
filter
Returns
Spare categories will be filled with Double.MaxValue.
Exceptions
IllegalArgumentExceptionif the length of the filter does not coincide with the number of categories in DistanceCategory.
