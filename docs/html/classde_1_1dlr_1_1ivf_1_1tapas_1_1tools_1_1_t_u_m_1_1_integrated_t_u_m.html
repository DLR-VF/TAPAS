Detailed Description

This class handles the integrated TUM-export. Additionally it extends SwingWorker

Input: since the constructor doesn't take any arguments this class gets its input through an ITumInterface interface that must be passed by the setInput(ITumInterface) method.

Work flow:

  • first a trip table will be created after DBTripReader constructor's call.
  • then you can access every TapasTrip by its implemented Iterator calling DBTripReader#getIterator().
  • every single trip can be passed as argument to GeneralAnalyzer#prepare(String, TapasTrip, de.dlr.ivf.tapas.util.parameters.TPS_ParameterClass) for evaluation.
  • finally an excel-export will be done by invoking GeneralAnalyzer#finish(File)

Note: file handling is done outside of this class for example in a customized JFileChooser like SimulationMonitor::tumFileChooser(int).

Thrown exceptions

  • PSQLException: thrown if IntegratedTUM#cancel(boolean) is invoked during the trip table creation statement and sometimes occurs during cancellation in a GeneralAnalyzer#prepare(String, TapasTrip, de.dlr.ivf.tapas.util.parameters.TPS_ParameterClass) call.
  • IllegalArgumentException: manually thrown by getAnalyzer(Categories) if no AnalyzerBase object that matched the Categories argument was found.
