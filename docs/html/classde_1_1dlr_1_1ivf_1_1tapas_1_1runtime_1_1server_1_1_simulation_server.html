Detailed Description

The main method of this class is used to setup a TAPAS remote server. Each machine which is registered in the database in the public.sim_servers table and where the server is running there can be remote simulations for tapas started.

staticprivate

Prevent RunWhenShuttingDown#run() from being executed twice when shutting down a server from the GUI. {@link #killServer()} invokes the RunWhenShuttingDown#run() procedure, which, when it returns, lets the main(String[]) finish. As RunWhenShuttingDown also being a ShutDownHook it will be invoked when main(String[]) finishes.
