Detailed Description

This class provides a scheme. All schemes are singleton instances, so that the plans have just a reference to the scheme but doesn't manipulate it. A scheme is divided into tour and home parts. Tour parts consists of stays and trips and home parts does only contain of stays which are located at home.

boolean de.dlr.ivf.tapas.scheme.TPS_Scheme.addEpisode ( TPS_Episode  episode)

This method adds the given episode to the last scheme part of the scheme. When there is no scheme part a home part is created (a scheme starts at home). When the last scheme past does not fit to the tour number or the fact that it is a home part a new part is created. At last the episode is added to the last scheme part.
