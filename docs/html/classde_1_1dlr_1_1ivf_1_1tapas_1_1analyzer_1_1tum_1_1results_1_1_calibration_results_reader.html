HashMap<Mode, Double> de.dlr.ivf.tapas.analyzer.tum.results.CalibrationResultsReader.getModalSplit ( Enum...  categories)

This method creates a modal split along some categories (or none). An example would be the modal split of shopping in 3-5 km. It returns the percentage of all trips (count) in the given category for each mode.

Parameters
categoriesmay be a combination of elements of PersonGroup, TripIntention and DistanceCategory (or DistanceCategoryDefault). Contradictory elements will result in empty maps.
Returns
null if the connection to the database went wrong.
Exceptions
IllegalArgumentExceptionif at least one of the categories are not valid or not suitable for modal splits.
