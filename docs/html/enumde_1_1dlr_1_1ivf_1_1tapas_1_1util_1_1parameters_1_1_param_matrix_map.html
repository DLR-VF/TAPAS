Detailed Description

This class provides all flag (boolean) enums which determine the name of parameters available in the application

This enum provides constants in a map of matrix form for the application. It is possible that on constant has two matrix maps depending on the type of the constant. If it is simulation type dependent there exist one matrix map for the base an one for the scenario case.
