Detailed Description

Ein Manager der sich um das Speihern der zuletzt selektierten Dateipfade kümmert. Wird der FileChooser oder FilesChooser verwendet, speichern diese den selektierten Pfad mithilfe dieser Klasse in einer Property-Datei. Beim nächsten Start der Anwendung wird diese Datei dann wieder ausgelesen und dient der Initialisierung der FileChooser bzw. FilesChooser
