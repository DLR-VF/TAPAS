◆ updateHouseholds()

void de.dlr.ivf.tapas.tools.OpportunityDistribution.HouseholdDistribution.updateHouseholds ( boolean  verbose)

Updates all household positions from block centroid to actual buildings. The households are distributed mainly on residential buildings. 3 stage algorithm:

  1. Try to match dlm blocks to blocks and find buildings in there
  2. If that fails, try find buildings just in blocks
  3. If that fails as well, update households to block centroid instead
Parameters
verboseIf true, a status report for every block is given to track progress.
