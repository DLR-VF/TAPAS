Detailed Description

Class for calculating the modal distributions in the case of a scenario. Based on the empirical modal splits as pivot point, the new distributions are calculated determining the differences in utility for the different modes available. Therefore, the changes in costs (financial and time-based) associated with the choice of a mode in the scenario and the base scenario are calculated.
