Detailed Description

This immutable object to collect a model and reference value, a label for this data point and its Quality. Quality is assigned by relative error of the data or provided on creation.

The Quality is set according to the relative error

  • Quality#GOOD if abs(ref-mod)/ref| < 0.1
  • Quality#BAD if abs(ref-mod)/ref| > 0.2
  • Quality#MEDIUM else

ref=0 results in Quality#MEDIUM.
