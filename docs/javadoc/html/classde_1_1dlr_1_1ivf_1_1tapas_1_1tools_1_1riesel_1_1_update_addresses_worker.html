Detailed Description

This class updates the number of inhabitants of the given list of addresses. The input of the data is handled by the BlockingQueue given and construction time.

The update is done in batches of size UpdateAddressesWorker#BATCH_MAX_SIZE. The last batch can be emptied by notifying this object via TPS_Main.TPS_State#setFinished().
