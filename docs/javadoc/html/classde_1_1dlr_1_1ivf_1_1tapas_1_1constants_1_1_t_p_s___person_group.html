Detailed Description

This class represents the person groups of TAPAS. Each person group contains of one sex, one car constant and some age classes. The age classes have a finer granularity than the person groups, so a person group have to store more than one age class. There also exist a fits*( method to check whether a person fits to the person group constant.

This method checks the following attributes:
a) TPS_AgeClass: Checks if the person's age class is included in the person group
b) TPS_Sex: checks the sex of person and group
c) TPS_CarCode: checks whether the amount of cars in the person's household corresponds with the constant in car d) TPS_HasChildCode checks whether the number of children in the person's household corresponds to the HasChildCode constant in the person group constant.
