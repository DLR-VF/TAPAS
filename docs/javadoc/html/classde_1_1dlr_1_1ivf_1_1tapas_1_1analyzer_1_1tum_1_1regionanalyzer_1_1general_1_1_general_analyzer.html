Detailed Description

This is the general logic module for the TUM export of the TAPAS Analyzer. It takes a tree of AnalyzerCollections and fills it accordingly. The main loop of adding TapasTrips is done outside this class. The method GeneralAnalyzer#prepare(String, TapasTrip, TPS_ParameterClass) is expected to be called very often.

The logic of this module is mostly handled by the Analyzers hold (and implicitly defined with the tree passed to the constructor) and the exporters. This class provides an entry point to the TAPAS analyzer structure.

As per the definition of the extended AbstractCoreProcess the method GeneralAnalyzer#finish() will trigger the default export methods. An excel export via GeneralExcelExport will be attempted. If the boolean writeToDB is set on construction, a GeneralDatabaseSummaryExport will be started as well.
