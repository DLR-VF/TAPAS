Detailed Description

This class provides basic methods to open and read from a comma separated value file via a CsvReader. You can receive the headers and each record, i.e. a line in the file. A method to check the header is also provided

important The method for checking the headers is not implemented, therefore it is depreciated. If you implement it, remove this comment and the deprecation statement.
